# ✦ YuiChyanBot 优衣酱 ✦

一个基于 Windows 平台的自用 QQ 机器人框架，基于 NoneBot 1.x + OneBot V11 协议，部分架构设计思想参考自 [HoshinoBot](https://github.com/Ice9Coffee/HoshinoBot)。
