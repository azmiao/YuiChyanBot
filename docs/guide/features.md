# 基础功能说明

框架自带三个核心插件，提供基础功能。每个插件目录下的 `HELP.md` 会被框架自动读取，生成帮助菜单图片和帮助网页。

## 基础功能（basic）

基础娱乐功能合集。

| 功能命令 | 介绍 |
|:---------|:-----|
| 查询授权 | 查询本群授权状态 |
| @BOT戳一戳我 | 让 BOT 戳一戳你 |
| {戳一戳BOT} | 无命令，戳一戳 BOT 触发消息 |
| @BOT翻译 en zh {文本} | 翻译文本 |
| @BOT文字识别 {图片} | 识别图片中文字 |
| @BOT漫画翻译 {图片} | 翻译漫画图片 |
| @BOT选择xxx还是yyy还是zzz | 帮助选择困难症选择 |
| @BOT tql是啥 | 帮你分析抽象拼音缩写是啥意思 |
| 掷骰子 1d6 | 掷骰子，例如掷 1 次 6 面的骰子 |
| 获取b站封面 xxx | xxx 为 b 站视频地址、av 号或直播 id |
| 解析二维码 {图片} | 解析二维码图片的 URL 地址 |
| 参与抽奖 | 参与本群当前已有的抽奖 |
| 查询抽奖 | 查询本群当前的抽奖情况 |
| 创建抽奖 {名字} {数量} | 【群管理】创建抽奖 |
| 结束抽奖 {名字} | 【群管理】结束抽奖 |

> 部分功能依赖外部 API，不稳定是正常现象。

## 核心管理（manager）

服务管理、授权管理、广播和帮助系统。

| 功能命令 | 介绍 |
|:---------|:-----|
| @BOT服务列表 | 【群管理】查询本群所有可见的服务 |
| @BOT启用服务 | 【群管理】对本群启用指定的服务 |
| @BOT禁用服务 | 【群管理】对本群禁用指定的服务 |
| 授权列表 | 【维护组私聊】查询所有授权列表 |
| 好友列表 | 【维护组私聊】查询所有好友列表 |
| 变更授权 | 【维护组私聊】变更某个群的授权 |
| 检查授权 | 【维护组私聊】手动检查一次授权状态 |
| 广播 g-13131313 广播内容 | 【维护组私聊】广播消息 |
| 退群 123456 管理员操作 | 【维护组私聊】要求 BOT 退群 |

## XQA 你问我答（xqa）

支持正则回流的高级问答系统。

### 普通问答命令

| 功能命令 | 介绍 |
|:---------|:-----|
| 我问A你答B | 设置个人问题 |
| 有人问C你答D | 【群管理】设置全群问答 |
| 不要回答H | 删除某个回答 H |
| 看看有人问 | 查看全群设置的问题 |
| 看看有人问X | 搜索全群设置的问题 |
| 看看我问 | 查看自己设置的问题 |
| 看看我问Y | 搜索自己设置的问题 |
| 查问答@某人 | 【群管理】查某人的全部问答 |
| 查问答@某人G | 【群管理】搜索某人的问答 |
| @某人不要回答H | 【群管理】删除某人的某个回答 H |

### 维护组管理命令

| 功能命令 | 介绍 |
|:---------|:-----|
| 全群问E你答F | 【维护组】设置所有群都回答的内容 |
| 看看全群问M | 【维护组】搜索所有群的有人问 |
| 全群不要回答N | 【维护组】在每个群的有人问里都删除某个问答 |
| 复制问答from群号1to群号2 | 【维护组】将群号1的有人问复制到群号2 |
| 复制问答from群号1to群号2-self | 【维护组】将群号1的个人问答复制到群号2 |
| 复制问答from群号1to群号2-full | 【维护组】将群号1的全部问答复制到群号2 |
| XQA禁用我问 | 【维护组】禁用该群的个人问答功能 |
| XQA启用我问 | 【维护组】启用该群的个人问答功能 |
| XQA清空本群所有我问 | 【维护组】清空该群所有人设置的我问 |
| XQA清空本群所有有人问 | 【维护组】清空该群的有人问 |
| XQA提取数据 | 【维护组】将数据提取成 xqa_db.json 文件 |
| XQA重建数据 | 【维护组】将 xqa_db.json 再生成 xqa_temp.db 后手动替换 |
